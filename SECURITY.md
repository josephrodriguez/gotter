## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Gotter, please report it responsibly.

**Do not** open a public GitHub issue for security vulnerabilities.

Instead, please email your report to:
```
gotter@gmail.com
```

Please include the following information in your report:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested fix (if available)

**Response Timeline:**
- We will acknowledge your report within 48 hours
- We will provide an initial assessment within 5 business days
- We will work on a fix and release a patch as soon as possible
- We will credit you for your responsible disclosure (if desired)

Thank you for helping keep Gotter secure!
